==== Gulp ====
O Gulp é um kit de ferramentas que ajuda a automatizar tarefas chatas ou demoradas no desenvolvimento
de seu projeto.("Nas aulas foi usada a versão ddo gulp 4")
LOADERS - É a estrutura usada no gulpfile para mover ou modificar os arquivos

=== INSTALAÇÃO DO GULP NO TERMINAL ===
npm i -g gulp-cli
gulp --version Para ver a versão
gulp --help Para ajuda

=== FUNDAMENTOS DO GULP 1 E 2 ====
Não esquecer de entrar na pasta basico para executar o comando de gulp e encontrar o arquivo gulpfile.js e realizar
a cópia dos arquivos TXT da pastaA para a pastaB

INTERFACE DE LINHA DE COMANDOS - cli
Inteface de linha de comandos que nos ajuda a trabalhar com as linguagens, frameworks etc...

=== Babel ===
Pega nosso código moderno e deixa o mais compativel possivel para os browsers

=== TypeScript ===
TypeScript é um superconjunto de JavaScript desenvolvido pela Microsoft que adiciona tipagem e alguns
outros recursos a linguagem.Não é interpretada pelo node, mais por exemplo com gulp conseguimos fazer um processo de 
conversão para javascript e assim ser interpretado normalmente pelo node.

=== Sass ===
É um pré-processador Css, que disponibiliza mais funcionalidades que não estão disponiveis no Css.

=== Spa - Single Page Application ===
As aplicações e os sites SPA, fazem uma transição entre os templates carregados, sem reload de página e sem que o usuário 
viaje de uma página para outra.

=== LEMBRETES ===
No spa, no arquivo deps.js o caminho encontramos dentro da pasta node_modules na pasta font-awesome, depois Css, ir 
no primeiro arquivo, depois com direito ir em "copy relative path".
