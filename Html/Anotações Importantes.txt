============================== LEMBRETES =============================== 

Sempre que startar o servidor colocar http-server -c-1 . para que ele não armazene arquivos, e rode como nodemon sempre
que fizermos mudanças em algum arquivo ele atualize automaticamente no projeto em questão.

COMANDO PARA COMENTAR VARIAS LINHAS ctrl + ; ou ctrl + /
COMANDO PARA MOVER LINHAS alt + cima ou baixo
COMANDO PARA COPIAR LINHA PARA BAIXO alt + shift + baixo
COMANDO PARA EDITAR PALAVRAS IGUAIS ctrl + D na palavra

Versão do express e body-parser usada no backend do conteudo
express@4.16.3 body-parser@1.18.2 -E para que baixe exatamente essa versão

Se fizermos uma requisição GET ele nem vem para o body (req.body) e sim (req.query)
